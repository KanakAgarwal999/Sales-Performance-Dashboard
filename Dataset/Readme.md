# Dataset Information

This folder contains the dataset used to develop the dashboard.

## Dataset

### Superstore Sales.xlsx

The dataset includes information related to:

- Orders
- Customers
- Products
- Categories
- Regions
- Shipping Modes
- Sales Revenue

## Data Fields

Examples:

- Order ID
- Order Date
- Customer Name
- Segment
- Product Name
- Category
- Sub-Category
- Sales
- Region
- State
- City
- Ship Mode

## Purpose

The dataset is used to analyze:

- Revenue Trends
- Customer Behavior
- Product Performance
- Regional Performance
- Shipping Efficiency
